# OMR Sheet Generator

This project generates print-ready A4 OMR bubble sheets as PDF files.

Current sheet features:
- 4 corner alignment markers plus local answer-area markers for grading registration
- 5-digit student ID area with `5` columns and digit bubbles `0-9`
- Question bubbles driven by an explicit per-question option-count list
- Up to `10` questions per column
- Variable answer counts per question from `2` to `5`
- Reserved QR code area at the top-right
- Automatic pagination when questions exceed one page

## Requirements

- Python `3.12+`
- [`uv`](https://docs.astral.sh/uv/)

## Setup With `uv`

Create the environment and install dependencies:

```bash
uv sync
```

## Generate a Sheet

Run the CLI with a comma-separated list of per-question option counts:

```bash
uv run omr --questions 4,4,5,3,2,5 --output omr-sheet.pdf
```

You can also override the title and instructions:

```bash
uv run omr \
  --questions 4,4,4,4,5,5,3,2 \
  --output exam-a.pdf \
  --title "Midterm Exam A" \
  --instructions "Fill bubbles completely. Use a dark pencil."
```

## CLI Reference

```bash
uv run omr --help
uv run omr-grade --help
```

Arguments:
- `--questions`: required comma-separated option counts, for example `4,4,5,3`
- `--output`: output PDF path, default `omr-sheet.pdf`
- `--title`: optional page title
- `--instructions`: optional instruction text

Grading:

```bash
uv run omr-grade omr-sheet-answered.pdf
```

Batch grading:

```bash
uv run omr-grade answered-sheets/
```

The grading command prints JSON with:
- `qr_data`
- `student_id`
- `marked_answers`
- `omr_error`

When you pass a folder, it reads all `.pdf` files in that folder and returns one JSON object per student file, including `source_pdf`. If one PDF fails, grading continues for the rest; that entry gets an `omr_error` message and empty `student_id` / `marked_answers`.

## Notes

- Each question count must be between `2` and `5`
- The default instructions text is `Fill bubbles completely.`
- Grading now requires the marker-based sheet format generated by the current version of this repo
